## 🚀 Quick Start After Download

1. **Download** the `ram-cycle-mart-export.zip` file
2. **Extract** on your web server
3. **Run** the installation script:
   ```bash
   ./install.sh
   ```
4. **Configure** your database in `.env.production`
5. **Build** and start:
   ```bash
   npm run build
   npm start
   ```
